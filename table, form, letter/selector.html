1. Class Selectors

It is represents by the symbol dot[.]. First need to put dot[.] then class name.

<div class="first"> </div>

Example: .first{styles}

2. ID selectors

It is represents by the symbol hashtag[#]. First need to put hashtag[#] then id name.

<div id="firstid"> </div>

Example: #firstid{styles}

3. Universal selectors

It is represents by the symbol star[]. First need to put star[] then style it. It allows yoy to select and style all elements. It is a css reset.

Example: *{styles}

3. Attribute selectors

It allows you to target elements based on their specific attribute.

<input type="text" id="fname" name="fname" placeholer="firstname">

Example: input[name="fname"]{styles}

4.Nested selectors

It allows you to nest one style rule inside another.

<body> <p> This is a <a href="https://ide.geeksforgeeks.org/">link</a> within a paragraph element. </p> <table style="width: 100%"> <tr> <th>Name</th> <th>Age</th> </tr>

Example: p a { color: green; } table tr th { background-color: beige; }

5.Grouping selectors

We can group them and write like this & we need the comma(,) to group the various selectors.

<body> <h1>GeeksForGeeks</h1> <h2>Smaller heading!</h2> <p>This is <a href="https://ide.geeksforgeeks.org/"> anchor tag </a> </p> <p>This is a paragraph.</p> </body>

Example: h1, h2, p, a { text-align: center; color: green; }

6.combinator selectors

There are some types of combinator seletors they are:

  • Descendant Combinator[space]
  • Adjacent Combinator[+]
  • Child Combinator[>]
  • Sibling Combinator[~]

I.Descendant combinator[space]:

It represents by [space]. It styles everthing within the tag name. For selecting elements that are descendants of other elements. First we need to put tage name then space.

<p>I am the first para</p> <div class='container'> <p>I am the second para</p> <div> <p>I am the third para</p> </div> </div> <p>I am the last para</p>

Example: .container p { color: red; }

The result for the above one will be it changes the color only to I am the second para and I am third para

II.Adjacent Combinator[+]:

It represents by symbol add[+]. For selecting elements that are immediate siblings after other elements. First we need to put tag name then add[+].

<p>I am the first para</p> <div class='container'> <p>I am the second para</p> <div> <p>I am the third para</p> </div> </div> <p>I am the last para</p> <p>I am the very last para</p>

Example: div + p { color: red; }

The result for the above one will be it changes the color only to I am the last para

III.Child Combinator[>]:

It represents by symbol Greater[>]. For selecting elements that are direct children of other elements. First we need to put tag name then Greater[>].

<p>I am the first para</p> <div class='container'> <p>I am the second para</p> <div> <p>I am the third para</p> </div> </div> <p>I am the last para</p>

Example: .container > div > p { color: red; }

The result for the above one will be it changes the color only to I am the third para

IV.Sibling Combinator[~]:

It represents by symbol tilde[~]. For selecting elements that are siblings after other elements. First we need to put tag name then tilde [~].

<p>I am the first para</p> <div class='container'> <p>I am the second para</p> <div> <p>I am the third para</p> </div> </div> <p>I am the last para</p> <p>I am the very last par</p>

Example: div ~ p { color: red; }

The result for the above one will be it changes the color only to I am the last para and I am very last para
